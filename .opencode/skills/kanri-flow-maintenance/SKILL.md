## メンテナンススキル

### 概要
アプリケーションの安定稼働とパフォーマンス維持を行います。

### 定期メンテナンス

#### 週次チェック
- [ ] 外部ライブラリの更新確認
- [ ] リンク切れチェック
- [ ] エラーログ確認

#### 月次チェック
- [ ] パフォーマンス測定
- [ ] セキュリティ脆弱性確認
- [ ] ブラウザ互換性テスト

### パフォーマンス最適化

#### 最適化項目
1. **CSS**: 不要なセレクタの削除、重複ルールの統合
2. **JavaScript**: 不要なコードの削除、最適化
3. **画像**: 圧縮、適切なフォーマット使用
4. **フォント**: 必要な文字のみロード

#### パフォーマンス確認ツール
- Google PageSpeed Insights
- Lighthouse
- ブラウザの開発者ツール

### セキュリティ確認

#### 確認項目
- [ ] XSS脆弱性の確認
- [ ] 入力値のサニタイズ
- [ ] 外部スクリプトの信頼性確認
- [ ] HTTPS通信の確認

#### セキュリティ対策
- ユーザー入力は必ずエスケープ
- 外部ライブラリは信頼できるソースから使用
- 定期的なセキュリティ更新

### メンテナンス記録
```markdown
## メンテナンス記録 ({日付})
### 実施内容
- {実施した内容}

### 結果
- {結果の詳細}

### 発見した問題
- {問題点}

### 対応策
- {対応策}
```

### 使用時のポイント
- 定期的にメンテナンスを実施
- 問題が発見された場合は早めに対応
- メンテナンス記録を残す
