## 品質管理スキル

### 概要
全ページのデザイン・リンク・機能が統一されているかを確認します。

### チェック項目

#### 1. デザイン一貫性チェック
- [ ] カラーパレットが統一されているか
- [ ] フォント設定が同じか
- [ ] 角丸サイズが適切か
- [ ] シャドウスタイルが統一されているか
- [ ] アニメーションが正しく動作するか

#### 2. リンク検証
- [ ] 全ページの遷移先URLが正しいか
- [ ] 404エラーがないか
- [ ] 外部リンクが有効か

#### 3. レスポンシブ検証
- [ ] モバイル表示が崩れていないか
- [ ] タップ領域が適切なサイズか
- [ ] フォントサイズが読みやすいか

#### 4. 機能検証
- [ ] テーマ切替が正しく動作するか
- [ ] フォーム送信が可能か
- [ ] ボタンクリックで正しいページに遷移するか

### 実行手順
1. 全ページを読み込み、CSS変数を確認
2. ブラウザの開発者ツールでモバイル表示を確認
3. 全リンクをクリックして遷移を確認
4. テーマ切替機能を全ページでテスト

### 結果記録
チェック結果は以下のように記録します：
- **OK**: 正常に動作
- **要修正**: 修正が必要
- **該当なし**: このページには適用されない
